# Trabalho Real Work Testing
Esse projeto usa Cypress para automatizar feature de login , cadastro e tranferencias bancarias.  O Cypress é um framework de testes ponta a ponta baseado em JavaScript que permite aos desenvolvedores escrever testes e executá-los em um navegador real.

## intallation
```bash
npm istall
```
>NOTE:
>
> É Necessario ter o **Node** instalado

## Running

```bash
# Clonar repositório
git clone https://github.com/seu-usuario/RealWorkTesting

# You open the Cypress UI
npx cypress open

# Run the automation by command line
npx cypress run
```
## Objetivo do projeto

Obejtivo do projeto é Por em pratica todo o aprendizado do curso Guardião da Qualidade - LumeStack, fazendo testes automatizados das funcionalidades do programa.
Garantindo mais qualidade e agilidade nos processos de testes usando o Cypress
